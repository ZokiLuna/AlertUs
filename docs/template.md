# Template padrão do site

O sistema Bairro Seguro foi desenvolvido com o objetivo de melhorar a segurança da comunidade, permitindo que usuários visualizem ocorrências em tempo real e relatem problemas em sua região. O design segue um padrão moderno com tema escuro (dark mode), priorizando usabilidade, clareza e organização das informações.

## Design

O layout do sistema é organizado de forma intuitiva e moderna:

- O logo "Bairro Seguro" está localizado no canto superior esquerdo.
- O menu superior contém as opções "Sobre" e "Como funciona".
- No canto superior direito há um botão de login e um ícone de configurações.
- O sistema possui uma barra lateral com ícones de navegação (menu lateral).
- A página inicial apresenta:
  - Um campo de busca de bairros
  - Um mapa com ocorrências
  - Botões de ação como "Ver ocorrências" e "Relatar problema"
- O sistema possui telas de:
  - Login
  - Cadastro
  - Recuperação de senha
  - Perfil do usuário
  - Edição de perfil
  - Configurações
- Os formulários são centralizados e seguem um padrão visual consistente.
- O mapa é utilizado para exibir ocorrências em tempo real.
- Os botões principais são destacados para facilitar a interação do usuário.

## Cores

A paleta de cores utilizada no sistema é:

- Fundo principal da tela: #1B1027
- Fundo dos cards/formulários: #787870
- Cor principal (botões e destaques): #126315
- Cor secundária (variações): verde mais escuro
- Cor de alerta: #FF3434
- Texto principal: #FFFFFF
- Texto secundário: #D9D9D9
- Campos de formulário: #D9D9D9
- Bordas e detalhes: cinza


## Tipografia
O sistema utiliza fontes sem serifa para melhor leitura:

- Títulos: maiores e em negrito
- Textos: tamanho padrão com boa legibilidade
- Botões: texto em destaque (negrito)
- Formulários: texto claro sobre fundo escuro

Essa escolha garante uma interface limpa e organizada.

## Iconografia

O sistema utiliza ícones simples e intuitivos para facilitar a navegação:

- Ícone de busca (lupa)
- Ícone de usuário (perfil)
- Ícone de senha (cadeado)
- Ícone de configurações (engrenagem)
- Ícones de navegação lateral
- Ícones de alerta (ocorrências)
- Ícones de mapa/localização
